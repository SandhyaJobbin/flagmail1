# Flagmail Zone 1 Demo Questions

Zone 1 is now fixed for demo use. These five emails will appear in the same order every time the assessment is launched, and their clue order will also stay the same.

## Fixed Zone 1 sequence

| # | Email ID | From | Subject | Expected L1 | Expected L2 | Reasoning question |
|---|---|---|---|---|---|---|
| 1 | `E003` | Netflix | `Your Netflix membership renewal for March 2026` | Legitimate | Subscription Billing | `What is the strongest reason this should be tagged Legitimate?` |

## Facilitator notes

Use this order when rehearsing the demo:

1. Start with one clean legitimate example to set the baseline.
2. Move into an obvious brand-spoof phishing example.
3. Show the difference between junk marketing spam and phishing.
4. Reinforce credential/payment harvesting with the fake Chase reward.
5. End Zone 1 with an advance-fee fraud example to show escalation in financial risk.
